# Python Data Analytics Projects

This repository contains beginner-level Python projects I developed during my training at Knowledge Tech Hub as I began my journey into Data Analytics.

The goal of these projects is to practice using Python to model real-world financial and business scenarios through simple analytical programs.

## Projects

### 1. TuneTech Expense Tracker
A budgeting tool that:
- Accepts daily expense inputs
- Calculates total and average spending
- Checks whether spending exceeds the daily budget

### 2. SIGMA Bank Loan Eligibility Checker
A rule-based financial system that determines loan eligibility using:
- Customer age
- Monthly income
- Credit score
- Missed payment history

### 3. Proactive Risk Alert System
A simple financial risk classification system that categorizes users as:
- High Risk
- Medium Risk
- Low Risk  
based on credit score, income, and repayment behavior.

### 4. Business Data Analyzer System
A retail analytics tool that evaluates:
- Product sales performance
- Stock levels
- Sales targets
- Inventory status

## Skills Demonstrated
- Python fundamentals
- User input handling
- Lists and dictionaries
- Conditional logic
- Financial and business data analysis
